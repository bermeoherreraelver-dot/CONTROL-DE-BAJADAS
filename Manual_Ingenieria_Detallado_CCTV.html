2.2 Configuración Crítica del NVR

El sistema debe cumplir con la configuración HA (Failover):

  • RAID 6 Obligatorio: Cada NVR (de los 6 totales) debe configurarse con 8 discos de 6TB. Al aplicar RAID 6, se obtienen 36TB útiles por NVR.
  • Capacidad Total Útil: 6 Servidores NVR x 36TB = 216 TB Instalados.
  • Margen de Seguridad: 36% (Permite absorber picos de actividad mayores al 70% estimado).

III. CONFIGURACIÓN DE REDUNDANCIA 1+1 (HIGH AVAILABILITY)

Se implementará una arquitectura Activo-Pasivo bajo el software Dahua DSS Pro o equivalente conforme a EETT.

3.1 Sincronización de Base de Datos (Hot Standby)

El Servidor 1 (Maestro) envía cada transacción de base de datos al Servidor 2 (Esclavo) a través de un enlace dedicado de latido (Heartbeat). Se sincronizan:

  • Listado de cámaras y credenciales.
  • Logs de auditoría y alarmas.
  • Configuración de mapas y roles de usuario.

3.2 Conmutación de Error (Failover)

Si el Servidor Maestro falla, el Servidor Esclavo detecta la ausencia de señal en < 5 segundos y levanta los servicios utilizando la IP Virtual (VIP). Los NVRs se reconectan automáticamente al nuevo servidor activo.

IV. SISTEMA DE VIDEOWALL Y MATRIZ VIRTUAL

Se detallan las conexiones y la gestión para los 8 monitores industriales de 50".

4.1 Topología Daisy Chain (Cascada)

Para evitar llevar 8 cables HDMI desde el rack de video, se usará la conexión Display Port 1.2:

  1. La Estación de Trabajo de Matriz Virtual envía 4 flujos principales.
  2. Cada flujo alimenta 2 monitores conectados en serie (Out del primer monitor a In del segundo).
  3. El software de Matriz Virtual identifica el arreglo 4x2 como una sola "Pared de Video".

4.2 Gestión de Vistas

  • Vistas Multi-pantalla: El operador puede dividir un solo monitor de 50" en 16 o 32 cuadrículas (Sub-streams).
  • IVS Pop-up: Las alarmas de cruce de línea e intrusión se mostrarán automáticamente en el monitor central en tamaño completo ("Pop-up View").

V. PROCEDIMIENTO DE CONFIGURACIÓN PASO A PASO (OPCIÓN E)

  1. Paso 1: Inicializar NVRs en RAID 6 y asignarles IPs en el rango 10.0.10.3X.
  2. Paso 2: Configurar el Smart H.265+ en todos los perfiles de cámara para asegurar el cumplimiento del almacenamiento.
  3. Paso 3: Habilitar IVS (Analítica) con filtros de Humanos y Vehículos para cumplimiento de reducción de falsas alarmas (EETT pág. 120).
  4. Paso 4: Sincronizar los 3 Joysticks USB/IP para el control fluido de las 6 cámaras PTZ desde las estaciones de monitoreo.
